    <h1>🇩🇪 German Dictionary</h1>

    <p>
        A simple German-English dictionary web application built with
        <strong>Python</strong> and <strong>Streamlit</strong>.
    </p>

    <h2>Features</h2>
    <ul>
        <li>Search German words instantly</li>
        <li>English translations</li>
        <li>CEFR language levels (A1, A2, B1, etc.)</li>
        <li>Part of speech information</li>
        <li>German example sentences</li>
        <li>English translations of example sentences</li>
        <li>Clean and user-friendly interface</li>
    </ul>

    <h2>Technologies Used</h2>
    <ul>
        <li>Python</li>
        <li>Streamlit</li>
        <li>JSON</li>
    </ul>

    <h2>Installation</h2>

    <h3>Clone the Repository</h3>
    <pre>
git clone https://github.com/saitarunaditya/German-Thesaurus.git
cd German-Thesaurus
    </pre>

    <h3>Install Dependencies</h3>
    <pre>
pip install -r requirements.txt
    </pre>

    <h3>Run the Application</h3>
    <pre>
streamlit run main.py
    </pre>

    <h2>Dataset</h2>
    <p>
        The application uses a JSON dataset containing:
    </p>
    <ul>
        <li>German word</li>
        <li>English translation</li>
        <li>CEFR level</li>
        <li>Part of speech</li>
        <li>German example sentence</li>
        <li>English example sentence</li>
    </ul>
