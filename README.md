<h1>🇩🇪 German Dictionary</h1>

A simple German-English dictionary web application built with Python and Streamlit.

Features
Search German words instantly
English translations
CEFR language level (A1, A2, B1, etc.)
Part of speech information
German example sentences
English translations of example sentences
User-friendly interface
Technologies Used
Python
Streamlit
JSON
Installation

Clone the repository:

git clone https://github.com/saitarunaditya/German-Thesaurus.git
cd German-Thesaurus

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run main.py
Dataset

The dictionary data is stored in JSON format and contains:

German word
English translation
CEFR level
Part of speech
Example sentence in German
Example sentence in English
